Canada FIFA 26 Schedule Project

The demo GIF that was added shows the functionilty of the "Add to Favourites button. 

Users can use this button to add their favourite match to the watchlist page and they can also delete those matches from the watchlist page. 

There is also an alert preventing users from adding the same match multiple times.

These features improves user interactions and alings with our project plan; building a website to show Team Canada's matches at the upcoming World Cup.
